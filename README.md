# Minifinch CLI

## Installation

Clone this repo somewhere safe on your machine, cd into it, then run the following:

```
$ chmod 755 install.sh
$ ./install.sh
```

## Usage

Run the following command and follow the on screen instructions.

```
$ minifinch
```
